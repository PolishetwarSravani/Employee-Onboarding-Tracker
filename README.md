# Employee-Onboarding-Tracker :  Excel Automation Project

Overview:
This project automates and tracks the onboarding process of employees across multiple departments. It uses Microsoft Excel to manage task completion, calculate onboarding status, flag overdue tasks, and display progress through interactive dashboards.

Key Features:

- Automated “Completed” / “Pending” status calculation
- SLA-based Expected Completion Date and Delay Flag
- Conditional formatting for visual tracking (color-coded alerts)
- Dashboard with pivot tables, charts, and KPIs
- Overdue and On-Track summaries with department filters

Skills Demonstrated:

- Data accuracy and attention to detail
- Excel formulas (IF, AND, COUNTIF, TODAY)
- Data validation and conditional formatting
- Reporting and visualization using Pivot Tables
- Dashboard design and SLA tracking

Tools Used:

- Microsoft Excel
- Data Validation, Pivot Tables, Conditional Formatting
- Snipping Tool (for screenshots)

Use Case:
Simulates real-world onboarding tracking used in corporate teams (like Deloitte’s On/Offboarding Delivery Center), ensuring quality, compliance, and timely reporting.
